Comparison: DOM vs Canvas

DOM Renderer

  • ✅ Text selection works
  • ✅ Screen readers work
  • ✅ Browser font rendering
  • ✅ CSS hover/focus states
  • ✅ DevTools inspection
  • ⚠️ More DOM nodes
  • ⚠️ Slower for large updates

Canvas Renderer

  • ❌ No native text selection
  • ❌ No screen reader support
  • ✅ Pixel-perfect control
  • ✅ Single DOM element
  • ✅ Faster batch updates
  • ✅ Export to image
  • ⚠️ Blurry when zoomed
